# Text-based-Projects

1. Suitor Selection- implemented a linked list of nodes to determine the position a suitor would stand when there are n suitors.

2. Haunted House Escape- used references to instances of a Node class to where the user has to determine which direction to go to in order to escape in node L.

3. Sentiment Analysis- analyzes .txt files of reviews to find the amount of positive and negative words used and determine the review's overall interpretation of being positive or negative.
